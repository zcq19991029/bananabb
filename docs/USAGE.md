## 申请与安装

1. 打开 https://bananabb-license-center.bananabb-zcq19991029.workers.dev/ 并使用 GitHub 登录。
2. 填写系统平台、设备备注和非商业使用说明，等待管理员审批。

5. 安装器会自动打开设备激活页；登录同一 GitHub 账号并输入卡密。
