# Bananabb 更新日志

工具开发：**讨厌下雨的天空**

本文档只记录对使用者有影响的变化，不依赖 Git 提交历史。

## 1.5.1 — 2026-07-22

### 授权与发布

- 发布 GitHub 账号 + 卡密 + 单设备绑定的 Mac/Windows 安装包。
- 新增在线申请中心、管理员审批台、邮箱人工补发渠道和个性化水印编号。
- 审批台增加冻结、恢复、解绑设备与永久撤销；撤销在用户下次启动时生效。
- 取消共享解压密码与旧 AES256 发布包。

### 使用规则

- 每张卡密仅绑定一个 GitHub 账号和一台设备；一个账号可经审批取得多张卡密。
- 继续明确禁止商业使用、收费安装、转售和二次分发。
